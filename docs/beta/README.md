# LockDigit Private Beta Operations

This directory defines the operating baseline for the 90-day private beta. All documents live in `docs/beta/`:

- `release-train.md` — Release train and freeze controls
- `incident-severity-and-response.md` — Incident severity and response standards
- `migration-runbook.md` — Schema migration and deployment
- `rollback-runbook.md` — Rollback procedures
- `beta-operating-limits.md` — Beta risk limits
- `monitoring-and-alerting.md` — Monitoring and alert targets
- `failure-injection-runbook.md` — Failure-injection drills
- `staged-rollout-runbook.md` — Staged rollout and operational cadence
- `launch-signoff-evidence.md` — Beta launch sign-off evidence template

These documents are enforced by CI checks in [`.github/workflows/ci.yml`](../../.github/workflows/ci.yml).
